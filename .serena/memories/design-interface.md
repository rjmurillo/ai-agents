# Skill-Design-006: Consistent Interface

**Statement**: All agents follow same input/output contract

**Context**: Agent template enforcement

**Evidence**: Template generation process enforces consistent structure

**Atomicity**: 85%

**Impact**: 7/10

## Detection

Inconsistent prompt formats

## Fix

Apply agent template strictly

## Standard Sections

1. Identity (who am I)
2. Purpose (what I do)
3. Entry Criteria (when to invoke)
4. Constraints (what I don't do)
5. Success Criteria (how to verify)
6. Handoff Protocol (next steps)
