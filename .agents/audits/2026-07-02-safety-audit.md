# Read-Only Architecture and Safety Audit

**Date**: 2026-07-02
**Branch**: `claude/ai-agents-safety-audit-x1scmo`
**Scope**: Full repository, read-only. Six passes: Python correctness (`scripts/`, `.claude/`), command execution and path safety, governance and mutability-boundary integrity, orphan/duplication candidates, subprocess and CI robustness.
**Method**: Six parallel audit agents, each grep-then-read confirmation. Every finding cites `path:line` plus an evidence quote. No fixes were applied; every recommendation below is a proposal.

## Premise corrections discovered during the audit

- `src/claude/*.md` agents are hand-maintained, NOT generated from `templates/agents/*.shared.md`. Only `src/vs-code-agents/` and `src/copilot-cli/agents/` are template-generated (evidence: `build/scripts/detect_agent_drift.py:1-25` "Claude agents have unique content and are NOT generated from templates").
- Skills, agents, and hooks dispatch by string name. Every orphan candidate in Pass E is therefore UNVERIFIED and requires human confirmation before any removal.

## Verified-clean areas

- CWE-78 command injection: no `shell=True`, `os.system`, or string-built commands in production code. All git/gh calls use list argv. No `Invoke-Expression` in any PowerShell. (Intentional vulnerable samples under `.agents/security/benchmarks/` excluded as fixtures.)
- Secrets: no committed tokens, keys, or credentials outside test placeholder data.
- Weak randomness: both `random.uniform` uses are backoff jitter, non-security.
- GitHub Actions pinning: every `uses:` in `.github/workflows/` and `.github/actions/` is a 40-char SHA or a local path. Zero unpinned references.
- Template/generated drift: `build_all.py --check` and `generate_agents.py --validate` both pass; all 21 commits touching `src/**` since 2026-01-01 are drift-clean.
- Instruction-tree mirrors: all 30 rules x 3 trees byte-identical after frontmatter strip. Zero drift.
- Return-type consistency: AST scan of all 264 `scripts/` modules found zero functions mixing value/None returns against their annotation.
